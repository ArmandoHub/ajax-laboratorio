# Selector Seleccionado Ejemplo
1 $("*") Selecciona todos los elementos
2 $("#") Selecciona un elemento que se identifique con un ID
3 $(".") Selecciona todos los elementos de clase
4 $("div") Selecciona todos los elementos de tipo etiqueta html
5 $("p") Selecciona todos los elementos de tipo parrafo
6 $("input") Selecciona todos los elementos de tipo input
